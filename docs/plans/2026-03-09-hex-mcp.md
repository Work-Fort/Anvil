# MCP Handler Cleanup Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** MCP handlers delegate to domain functions instead of reimplementing logic with
direct filesystem access.

**Architecture:** After Phases 3-7 give domain functions clean APIs that accept paths and
return data, MCP handlers become thin adapters: parse request → call domain → serialize
response. No more `os.ReadDir`, `os.Readlink`, `os.RemoveAll` in handler code.

**Tech Stack:** Go, mcp-go

### Task 1: Rewrite `handleKernelList` to use domain

**Files:**
- Modify: `internal/mcp/tools_kernel_mgmt.go`

**Step 1: Replace filesystem access with domain call**

Current handler (lines 45-105) does `os.ReadDir`, `os.Readlink`, `filepath.Join` —
reimplementing what `kernel.List()` does.

After Phase 3, `kernel.List(paths)` returns `[]kernel.KernelInfo`. Use it:

```go
func handleKernelList(_ context.Context, req gomcp.CallToolRequest) (*gomcp.CallToolResult, error) {
	archFilter := req.GetString("arch", "")

	kernels, arch, err := kernel.List(config.GlobalPaths)
	if err != nil {
		return errResult(err)
	}

	// Apply arch filter if specified
	if archFilter != "" && archFilter != arch {
		return jsonResult(map[string]any{
			"arch":    archFilter,
			"kernels": []any{},
		})
	}

	return jsonResult(map[string]any{
		"arch":    arch,
		"kernels": kernels,
	})
}
```

### Task 2: Rewrite `handleKernelGet` to use domain

**Files:**
- Modify: `internal/mcp/tools_kernel_mgmt.go`

Current handler (lines 107-150) does `os.ReadDir`, `filepath.Join`, `os.Readlink` —
building kernel info manually.

After Phase 3, use a domain function to get single kernel info:

```go
func handleKernelGet(_ context.Context, req gomcp.CallToolRequest) (*gomcp.CallToolResult, error) {
	version, err := req.RequireString("version")
	if err != nil {
		return errResult(err)
	}

	kernels, _, err := kernel.List(config.GlobalPaths)
	if err != nil {
		return errResult(err)
	}

	for _, k := range kernels {
		if k.Version == version {
			return jsonResult(k)
		}
	}

	return errResult(fmt.Errorf("kernel version %s not found", version))
}
```
